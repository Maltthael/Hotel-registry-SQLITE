# Hotel Registry (SQLite)

A small and simple **user registration system for a hotel**, built with **Python** and **SQLite3**.

This project was created for learning purposes, focusing on basic database operations using `sqlite3`.

---

## Features

- Create a SQLite database
- Register users
- Read stored users
- Update user data

---

## Technologies

- Python 3
- SQLite3 (built-in Python module)
